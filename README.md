# Planning tot 30 april 2026

**Link:** [Open het board](https://phobiq.github.io/planning-kanban/)

### Hoe werkt het?
1. Voer je naam in → klik "Stemmen instellen"
2. Klik op een badge om te stemmen
3. Sleep kaartjes tussen kolommen
4. Klik op **Exporteer** → kies CSV, PNG of PDF

Werkt op telefoon, tablet en laptop!
